## نصائح مهمة

1. **دايماً اعمل `--dry-run` الأول** قبل ما تمسح أي حاجة
2. **الباك أب هيبقى أصغر بكتير** - من 100+ GB لـ 40 GB مثلاً
3. **الملفات هترجع تاني** - مجرد ما تفتح المشروع في Final Cut Pro
4. **السكربت آمن** - بيمسح Render Files و Analysis Files و Transcoded Media بس

## المقارنة مع ncdu

### ncdu (الطريقة القديمة):
```
1. ncdu /path/to/folder
2. انتظر التحليل (5-10 دقائق)
3. ادخل كل مشروع
4. ادخل فولدر التاريخ
5. احذف Analysis Files يدوياً
6. ارجع
7. احذف Render Files يدوياً
8. كرر لكل مشروع (8 مشاريع × 2-3 دقائق = 20+ دقيقة)
```

### fcp_cleaner (الطريقة الجديدة):
```
1. ./clean.sh --auto
2. انتهى! (30 ثانية)
```

**التوفير:**
- ⏱️ الوقت: من 20+ دقيقة → 30 ثانية
- 🎯 دقة: 100% (بيمسح الفولدرات الصح بس)
- 💾 المساحة: نفس النتيجة (60+ GB)

✓ تنقل بين المجلدات بالأسهم
✓ مسح انتقائي أو شامل
✓ رجوع للمتصفح في أي وقت (b)

🎯 دورة العمل الكاملة
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    ┌──────────────────┐
    │ Start Program    │
    └────────┬─────────┘
             │
             ↓
    ┌──────────────────┐
    │ File Browser     │ ← تصفح المجلدات
    └────────┬─────────┘
             │ (اضغط s)
             ↓
    ┌──────────────────┐
    │ Scan & Delete    │ ← مسح الملفات
    └────────┬─────────┘
             │
             ├─ (اضغط q) → خروج نهائي
             │
             ├─ (اضغط b) → رجوع ⤴ للمتصفح
             │
             └─ (خلصت كل حاجة) ┐
                                │
                    ┌───────────┴──────────┐
                    │ All Done!            │
                    │ b: Go back           │
                    │ q: Quit              │
                    └───────┬──────────────┘
                            │
                            ├─ (اضغط b) → رجوع ⤴ للمتصفح
                            │
                            └─ (اضغط q) → خروج

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 سيناريوهات الاستخدام
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  مسح من مجلد واحد:
    Browse → Scan → Delete → q

2️⃣  مسح من عدة مجلدات:
    Browse → Scan → Delete → b → Browse → Scan → Delete → b...

3️⃣  غيرت رأيك أثناء المسح:
    Browse → Scan → b → Browse مسار جديد

4️⃣  خلصت مجلد، عايز تكمل مجلد تاني:
    Browse → Scan → Delete all → b → Browse → Scan...

⌨️  الاختصارات الكاملة
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

في المتصفح:
┌────────────────────────────────────┐
│ ↑/↓         - التنقل بين المجلدات │
│ ENTER       - دخول المجلد          │
│ BACKSPACE   - رجوع للمجلد الأعلى   │
│ s           - بدء المسح            │
│ q           - خروج نهائي           │
└────────────────────────────────────┘

في واجهة المسح:
┌────────────────────────────────────┐
│ ↑/↓         - التنقل بين الملفات  │
│ SPACE       - تحديد/إلغاء تحديد    │
│ d           - مسح الملف الحالي     │
│ D           - مسح كل الملفات       │
│ b           - رجوع للمتصفح         │
│ q           - خروج نهائي           │
└────────────────────────────────────┘

بعد الانتهاء:
┌────────────────────────────────────┐
│ b           - رجوع للمتصفح         │
│ q           - خروج نهائي           │
└────────────────────────────────────┘

مثال 1: مسح من 3 أشهر مختلفة
────────────────────────────────────
1. Browse → December → s → Delete all
2. Press b
3. Browse → November → s → Delete all
4. Press b
5. Browse → October → s → Delete all
6. Press q

✅ مسحت 3 أشهر في جلسة واحدة!

مثال 2: بدأت في مكان غلط
────────────────────────────────────
1. Browse → January → s
2. "إيه ده؟ أنا كنت عايز December!"
3. Press b
4. Browse → December → s → Delete
5. "تمام!"

✅ صححت المسار بسرعة!

مثال 3: عدة مشاريع في أماكن مختلفة
────────────────────────────────────
1. Browse → Desktop/Projects → s → Delete
2. Press b
3. Browse → Work/Videos → s → Delete
4. Press b
5. Browse → External/Backups → s → Delete
6. Press q

✅ نظفت 3 أماكن مختلفة!

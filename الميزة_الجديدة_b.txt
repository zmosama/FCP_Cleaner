🔄 دورة العمل الكاملة
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

     ┌──────────────┐
     │ File Browser │ ← تبدأ هنا
     └──────┬───────┘
            │ (اضغط s)
            ↓
     ┌──────────────┐
     │ Scan & Clean │ ← تمسح الملفات
     └──────┬───────┘
            │
            ├─ (اضغط q) → خروج
            │
            └─ (اضغط b) → رجوع للمتصفح ⤴

يعني تقدر تعمل loop:
  تصفح → مسح → رجوع → تصفح → مسح → رجوع...

لحد ما تخلص كل المجلدات اللي عايزها!

💡 حالات الاستخدام
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. عندك مشاريع في أماكن مختلفة:
   Desktop/2024 → [مسح] → b → Desktop/2023 → [مسح]

2. بدأت في مكان غلط:
   فتحت يناير بدل ديسمبر → b → اختار ديسمبر

3. عايز تمسح من مجلدات متعددة:
   Work/Projects → [مسح] → b → Personal/Videos → [مسح]

في المتصفح:
  ↑/↓         - التنقل
  ENTER       - دخول مجلد
  BACKSPACE   - رجوع للمجلد الأعلى
  s           - بدء المسح
  q           - خروج

في واجهة المسح:
  ↑/↓         - التنقل
  SPACE       - تحديد
  d           - مسح فولدر
  D           - مسح الكل
  b           - رجوع للمتصفح ← جديد!
  q           - خروج
